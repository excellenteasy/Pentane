Pentane

Pentane is a convenient way to access your Web Apps like native Apps.
Just without losing the benefits of their natural browser environment.

By now Chrome is the only supported browser. Others may follow. Pentane is a Beta version. Bugs may appear.

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/excellenteasy/Pentane
